- Compact Profiles
* Cada JVM necessita de subir as suas  classes básicas da API para a memória. Se em um computador tem mais de uma JVM, essas classes básicas não são duplicadas na memória, as JVMs usam um mesmo espaço de memória para armazenar as classes básicas, de modo que cada JVM apenas carregará na memória o que a cada aplicação precisará. No java8 foram incluídos perfis compactos (compact profiles) onde cada profile só sobe com classes específicas e é atribuído a cada JVM e  para economizer memória.
